经过一年半开发，Olares 团队在开发系统应用时已经逐渐迁移到 DevBox 上。

下面我们将演示如何在 Devbox 里调试 Desktop 的前端代码。

2. 配置 [deployment.yaml](./deployment)
3. 配置 [OlaresManifest.yaml](./olares-manifest)
4. 安装应用
5. 配置 [nginx.conf](./install)
